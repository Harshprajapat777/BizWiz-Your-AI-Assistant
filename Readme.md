# BizWiz ~ your AI assistance

It is the RAG based Chatbot which supports cleint query and give real time assistance.

It is build by the OpenAI LLM ,flask Server and Langchainframework.

# note -

1. buffered memory feature so that conversation is more focused.
2. token by token streaming reponse as OPENAI and gemeini and evry llm reponse
3. 500 chunk size and overlap 100 can be vary according to context data.

# Tools Used.

Research - comet and deepseek and claudecode opus 4.5.

# working.

1. install the dependency (pip install -r requirements.txt)
2. CORS connection frontend and backend connectivity
3. load the data context(Custom_data.txt)
4. created global varible (system memory to store the chunks and embeddings)
5. langchain functions (CHEL + text Splitter + emebedding modals + strOutputparser)
6. openAI and FAISS embedding
7. gpt3.5 turbo & temperatiure 0.7 ( mature response)
8. prompt engineering
9. LCEL CHAIN PARSER for request response
